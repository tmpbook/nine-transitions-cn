# Glossary（词汇表）

| 英文定义 | 中文定义 |
|---------|----------|
| 1chbo,1cb — The first break of a channel move. Usually fails and signals entry in channel direction. | 通道运动的第一次突破，通常会失败，是顺着通道方向入场的信号。 |
| 1tf,5tf — 1 tick failure; price goes 1 tick beyond prior bar then reverses and hits stop. | 1 跳失败。价格向前柱外多走 1 tick 后反转并打掉止损。5tf/9tf 同理。 |
| 1PB — First pullback of the day or after 1Rev. | 当天第一次回调，或 1Rev 之后的第一次回调。 |
| 1Rev — First reversal of the day. | 当天第一次反转。 |
| OR — Opening Reversal; reversal after early trend attempt. | 开盘反转：开盘第一小时趋势尝试后的反转。 |
| 1L,2L — 1-legged or 2-legged move. | 单腿或双腿走势。 |
| A2 — Second attempt to reverse a pullback near EMA; continuation signal. | 靠近 EMA 的第二次反转尝试，也是趋势延续信号。离 EMA 太远则不算。 |
| A22 — Second entry of A2 due to large stop or overlap. | A2 的第二次入场，通常因为第一次信号止损过大或重叠多。 |
| H1 — First bar to tick above previous bar in a down move. | 下跌中第一次向上突破前柱高点，首次反转尝试。 |
| fH1 — Price falls below the H1 bar, failing the reversal. | 价格跌破 H1 柱，反转失败。 |
| fH1 MM — Failed H1 in strong downtrend often leads to measured move. | 强下跌中失败的 H1 常引发测量移动（1:1）。 |
| H2 — Second bar to tick above previous bar after H1/fH1. | H1/fH1 后第二次向上突破前柱，高概率反转。 |
| L1,fL1,L2 — Opposite of H1,fH1,H2 in an up move. | 上涨趋势中 H1,fH1,H2 的反向版本。 |
| MM — Measured move, expected 1:1 projection. | 测量移动（1:1），常用突破柱长度作为目标。 |
| BO — Breakout beyond a price action barrier. | 突破关键结构（高点、EMA、趋势线等）。 |
| BP — Breakout pullback, price pulls back inside broken barrier. | 突破后的回调，价格回到被突破的结构内。 |
| fBO — Failed breakout. | 突破失败。大多第一次突破会失败。 |
| DP — Pullback entry after double bottom/top. | 双底后的多头回调入场，双顶后的空头入场。 |
| WP — Three-legged pullback, similar to A2. | 三腿回调，类似 A2。 |
| TL — Trendline connecting swing highs/lows. | 趋势线，连接上升低点或下降高点。 |
| TCL — Trend channel line parallel to TL. | 趋势通道线，与趋势线平行。 |
| G — Gap bar where low/high is beyond EMA in trend direction. | Gap 柱：下跌中高点低于 EMA；上涨中低点高于 EMA。 |
| G2 — Second G bar when first G fill fails; higher probability. | G 回补失败后出现的第二根 G，胜率更高。 |
| i, o — inside bar, outside bar. | 内包柱、外包柱。 |
| ii — inside bar inside another inside bar. | 内包中的内包（多层）。 |
| LL, HH — Lower Low, Higher High. | 更低的低点，更高的高点。 |
| mW — micro wedge; wedge formed by bars, not swings. | 微楔形，由柱组成的 TCL。 |
| PB — pullback. | 回调。 |
| oio, ud — outside-inside-outside or up-down reversals. | 外–内–外反转，上–下反转。 |
| TTR — Tight trading range with overlapping bars. | 窄幅横盘，多根重叠柱，来回突破少量 tick。 |
| BW — Barb wire: 3+ horizontal bars with a doji. | 铁丝网：三根以上横盘柱含一个十字星，类似 TTR。 |
| W — Wedge, three pushes with final overshoot. | 楔形：三段推动，第三段常超越趋势线，带来反转。 |
| W1P — First pullback after a W reversal. | W 反转后的第一次顺势回调入场。 |
